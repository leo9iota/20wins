A simple CLI game written in shell

### Game Rules
You play against a computer which picks either 1 or 2. You, as the player can also pick between 1 or 2. The first to reach 20 wins, as the name implies.

### Installation
```bash
chmod +x 20wins.sh
./20wins.sh
```
